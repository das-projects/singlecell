Profiling Scanpy for data of Zheng et al., Nat. Comm. (2017)¶

This is the Scanpy benchmark for the Cell Ranger R analysis of Zheng et al., Nat. Comm. (2017) available from https://github.com/10XGenomics/single-cell-3prime-paper. Compare the Scanpy version with the Cell Ranger version of this notebook.

The pbmc64k data is freely available [page/file] from the 10x homepage or from the previous GitHub link.
